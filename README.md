AVX PreflightPro is a browser-based print file validation platform that performs automated GRACoL 2013 CRPC6 preflight analysis on PDF and image files. It inspects color spaces, ink coverage (TAC), font embedding, image resolution, bleed, transparency, and ICC profile compliance — then surfaces issues as numbered markers overlaid directly on the document preview. Markers are clickable, opening a 200% magnified popup of the flagged region for precise visual review. The tool includes CMYK color separation views, a live eyedropper for GRACoL ink sampling, AI-powered font recognition, one-click automated fix-up, and a downloadable preflight report — all without leaving the browser.
